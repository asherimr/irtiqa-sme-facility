# The Irtiqa SME Facility

A structured, asset-backed Islamic financing ecosystem bridging Pakistan's Missing Middle.

## Overview

The Irtiqa SME Facility is a comprehensive proposal for Islamic SME financing that transitions unbanked SMEs from short-term liquidity (via Trade-Based Salam) to long-term growth (via Equity-Based Mudarabah) using a mechanism that eliminates bank risk exposure.

## Features

- **Structured Collateral**: Hamish Jiddiyah Lock mechanism
- **Parallel Salam**: Zero price risk for the bank
- **Growth Ladder**: Clear transition path to equity financing
- **Responsive Design**: Optimized for both mobile and web viewing
